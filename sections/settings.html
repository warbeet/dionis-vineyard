🇷🇺 Регион и локализация

Выберите регион виноградника, валюту, часовой пояс и провайдера карт. По умолчанию — Краснодарский край.

🔑 OpenRouter API (AI-анализ фото)

Получите ключ на openrouter.ai/keys. Поддерживает GPT-4o, Claude 3.5, Gemini и десятки других моделей одним ключом.

🤖 OpenRouter используется для: анализа фото, заполнения карточек препаратов/удобрений, AI-прогноза полива/болезней/работ на 7 дней.

🌐 Web-search инструкций препаратов

Ключ хранится локально в браузере. Получить: tavily.com

Endpoint должен принимать POST JSON { query, productName } и возвращать { results:[{title,url,content}] }.

🔥 Firebase (облачная база и синхронизация)

Firebase-проект Dionis vineyard уже встроен в приложение. Здесь можно посмотреть правила Firestore или при необходимости переопределить Firebase config для другого проекта. Sync V2 хранит данные чанками и не упирается в лимит 1 МБ одного документа.

🔐 Правила Firestore для Sync V2:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /vineyards/{vineyardId} {
      allow read, write: if request.auth != null;
      match /{document=**} {
        allow read, write: if request.auth != null;
      }
    }
    match /users/{uid} {
      allow read, write: if request.auth != null && request.auth.uid == uid;
    }
    match /invitations/{id} {
      allow read, write: if request.auth != null;
    }
    match /accessRequests/{uid} {
      allow read, write: if request.auth != null;
    }
  }
}
