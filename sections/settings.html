⚙️ Настройки

🇷🇺 Регион и локализация

Выберите регион виноградника, валюту, часовой пояс и провайдера карт. По умолчанию — Краснодарский край.

🔑 OpenRouter API (AI-анализ фото)

Получите ключ на openrouter.ai/keys. Поддерживает GPT-4o, Claude 3.5, Gemini и десятки других моделей одним ключом.

🟡 Авторизация через Яндекс ID

Войдите через свой Яндекс-аккаунт. Безопасно, без хранения пароля. Используется для идентификации команды.

🔥 Firebase (синхронизация)

Создайте проект на console.firebase.google.com, включите Authentication (Email + Google) и Firestore. Скопируйте конфиг из настроек проекта.

📱 PWA

Установите приложение на устройство для оффлайн-работы.

💡 Если кнопка не работает — используйте меню браузера: «Добавить на главный экран».
